### 完全新手(从未接触游戏开发)

1. **第1步**: [游戏开发技术栈全景图](./01-游戏开发技术栈全景图.md)

   - 理解各个技术的作用

2. **第2步**: [游戏开发学习路线](./06-游戏开发学习路线.md)

   - 知道从哪里开始

3. **第3步**: [图形API详解](./03-图形API详解.md)
   - 理解图形渲染基础
   - OpenGL 入门

4. **第4步**: 返回主知识库,学习 Soluna 项目

### 有编程基础(想了解云风的技术)

1. **第1步**: [云风技术演进史](./02-云风技术演进史.md)
   - 理解云风的技术选择逻辑
   - 了解各项目的定位

2. **第2步**: [游戏引擎架构](./04-游戏引擎架构.md)
   - 理解引擎设计理念
   - 对比商业引擎和自研引擎

3. **第3步**: 深入学习感兴趣的项目
   - Soluna(2D 游戏引擎)
   - skynet(游戏服务器)
   - Ant Engine(3D 引擎)

### 已有游戏开发经验(想深入理解)

- 直接阅读 [云风技术演进史](./02-云风技术演进史.md)

## 🎓 学习建议

### 推荐的学习顺序

```
阶段1: 基础知识
  ↓
阶段2: 图形编程入门(OpenGL/Sokol)
  ↓
阶段3: 2D游戏开发(Soluna/LÖVE)
  ↓
阶段4: 服务器开发(skynet)
  ↓
阶段5: 3D引擎开发(Ant Engine)
```

### 不同背景的学习路径

**Web开发背景**:
1. 先学图形API基础(OpenGL)
2. 学习C/C++(如果不熟悉)
3. 从2D游戏引擎开始(Soluna)
4. 逐步深入引擎架构

**C/C++开发背景**:
1. 直接学习图形API(OpenGL/Vulkan)
2. 研究Soluna源码
3. 尝试实现小型游戏
4. 深入引擎架构

**Python背景**:
1. 先用Pygame做小游戏
2. 学习C/C++基础
3. 转向Lua+C组合(Soluna)
4. 深入底层技术
