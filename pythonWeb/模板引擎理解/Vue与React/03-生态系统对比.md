# Vue vs React - 生态系统对比

## 1. 核心库对比

### 1.1 路由

#### Vue Router (官方)

```typescript
// 安装
npm install vue-router@4

// 配置
import { createRouter, createWebHistory } from 'vue-router'

const router = createRouter({
  history: createWebHistory(),
  routes: [
    { path: '/', component: Home },
    { path: '/about', component: About }
  ]
})

// 使用
app.use(router)
```

**特点:**
- ✅ 官方维护
- ✅ 与 Vue 深度集成
- ✅ 文档完善
- ✅ 中文文档

#### React Router (社区)

```typescript
// 安装
npm install react-router-dom

// 使用
import { BrowserRouter, Routes, Route } from 'react-router-dom'

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
      </Routes>
    </BrowserRouter>
  )
}
```

**特点:**
- ⚠️ 社区维护 (非官方)
- ✅ 功能强大
- ✅ 广泛使用
- ⚠️ API 变化较大

### 1.2 状态管理

#### Vue - Pinia (官方推荐)

```typescript
// 安装
npm install pinia

// 定义 Store
import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    name: 'Zhang San',
    age: 25
  }),
  getters: {
    doubleAge: (state) => state.age * 2
  },
  actions: {
    increment() {
      this.age++
    }
  }
})

// 使用
import { useUserStore } from '@/stores/user'
const userStore = useUserStore()
userStore.increment()
```

#### React - Redux Toolkit (主流)

```typescript
// 安装
npm install @reduxjs/toolkit react-redux

// 定义 Slice
import { createSlice } from '@reduxjs/toolkit'

const userSlice = createSlice({
  name: 'user',
  initialState: { name: 'Zhang San', age: 25 },
  reducers: {
    increment: (state) => {
      state.age += 1
    }
  }
})

// 使用
import { useDispatch, useSelector } from 'react-redux'
const dispatch = useDispatch()
const age = useSelector((state) => state.user.age)
dispatch(increment())
```

**其他选择:**
- Zustand (轻量级)
- MobX (响应式)
- Jotai (原子化)

### 1.3 HTTP 请求

#### 通用: Axios

```typescript
// 两个框架都常用
npm install axios

// Vue
import { ref } from 'vue'
import axios from 'axios'

const data = ref(null)
const fetchData = async () => {
  data.value = await axios.get('/api/data')
}

// React
import { useState } from 'react'
import axios from 'axios'

const [data, setData] = useState(null)
const fetchData = async () => {
  const result = await axios.get('/api/data')
  setData(result.data)
}
```

#### Vue: VueUse / axios

```typescript
// VueUse 提供的组合式函数
import { useFetch } from '@vueuse/core'

const { data, error, isFetching } = useFetch('/api/data').json()
```

#### React: SWR / React Query

```typescript
// SWR
import useSWR from 'swr'

const { data, error } = useSWR('/api/data', fetcher)

// React Query
import { useQuery } from '@tanstack/react-query'

const { data, isLoading } = useQuery({
  queryKey: ['data'],
  queryFn: fetchData
})
```

```bash

✅ 官方推荐
✅ 零配置 SSR
✅ 文件路由
✅ 自动导入

```

```vue
<!-- pages/index.vue -->
<script setup>
// 服务端获取数据
const { data: posts } = await useFetch('/api/posts')
</script>

<template>
  <div>
    <h1>文章列表</h1>
    <div v-for="post in posts" :key="post.id">
      {{ post.title }}
    </div>
  </div>
</template>
```

### React - Next.js

```bash
# 创建项目
npx create-next-app@latest

# 特点:
✅ 最流行
✅ SSR/SSG/ISR
✅ 文件路由
✅ API Routes
✅ 图片优化
```

```tsx
// app/page.tsx
async function getData() {
  const res = await fetch('https://api.example.com/posts')
  return res.json()
}

export default async function Home() {
  const posts = await getData()

  return (
    <div>
      <h1>文章列表</h1>
      {posts.map(post => (
        <div key={post.id}>{post.title}</div>
      ))}
    </div>
  )
}
```

## 10. 如何选择

### 选择 Vue 生态当:

```
✅ 需要官方完整工具链
✅ 中文文档很重要
✅ 团队前端基础一般
✅ 国内项目为主
✅ 快速开发优先
```

### 选择 React 生态当:

```
✅ 需要更多第三方库选择
✅ 移动端开发需求 (React Native)
✅ 大型企业级项目
✅ 团队 JavaScript 基础好
✅ 国际化项目
```

### 实际建议

```
小型项目:
└─ Vue (更快上手)

中型项目:
├─ Vue (如果团队熟悉)
└─ React (如果有经验)

大型项目:
├─ React (生态更成熟)
└─ Vue (如果已有技术栈)

移动应用:
└─ React (React Native)

桌面应用:
├─ Vue + Electron
└─ React + Electron
```

---

**总结: Vue 生态更集中(官方),React 生态更分散(社区),各有优势!**
