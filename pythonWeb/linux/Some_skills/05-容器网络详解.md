## 容器网络架构

```
容器 <--> veth <--> bridge <--> iptables <--> 外网
```

## veth pair

虚拟网卡对，数据从一端进另一端出。

```bash
sudo ip link add veth0 type veth peer name veth1
sudo ip addr add 192.168.1.1/24 dev veth0
sudo ip link set veth0 up
```

## Docker 网络

```bash
docker network create mynet
docker run --network mynet nginx
```
