## 完整容器构建

综合运用 Namespace + Cgroups + chroot + 网络

```bash
#!/bin/bash
# 1. 准备 rootfs
mkdir -p /tmp/container/{{bin,lib}}
cp /bin/bash /tmp/container/bin/

# 2. 创建容器
sudo unshare --pid --fork --mount --uts --ipc --net /bin/bash
```

## Python 实现

```python
import subprocess
subprocess.run(['unshare', '--pid', '--fork', '/bin/bash'])
```
