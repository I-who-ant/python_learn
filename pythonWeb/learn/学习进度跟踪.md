## 💡 关键知识点记录

### Flask 核心概念
```
1. 应用实例: Flask(__name__)
2. 路由: @app.route()
3. 视图函数: def view()
4. 请求: request 对象
5. 响应: response 对象
6. 模板: render_template()
7. ...
```

### 最佳实践
```
1. 使用虚拟环境
2. debug 模式仅用于开发
3. 密码必须哈希存储
4. 使用 CSRF 保护
5. ...
```

### 常见陷阱
```
1. 忘记激活虚拟环境
2. 循环导入问题
3. 数据库连接未关闭
4. ...
```
