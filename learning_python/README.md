# learning_python

## 仓库目的
- 构建一个面向自学的 Python 练习与笔记空间。
- 依托 Jupyter Notebook 记录学习过程、代码实验和心得。
- 强调小步迭代与及时总结，避免一次性堆过多内容（贯彻 KISS、YAGNI）。

## 目录概览
- `sample.ipynb`：当前的 Jupyter 学习笔记，涵盖控制流、函数定义与文件操作示例。
- `learn_progress.md`：每次指导或练习后的进度记录。
- `test.py` / `python.md`：临时脚本或补充笔记，可按需整理归档。
- `.venv/`：项目专用虚拟环境，隔离依赖。

## 使用方式
- 激活虚拟环境后运行 `python -m jupyterlab`（或安装 `notebook` 后运行 `jupyter notebook`）打开交互式笔记本。
- 鼓励将每次学习的关键代码片段保存在 `sample.ipynb`，并在 `learn_progress.md` 中补充背景、收获与待办。

## 维护约定
- 任何指导或学习活动完成后 **必须同步更新** `learn_progress.md`，写入日期、主题和下一步计划，以便追踪进展。
- 若新增文件或目录，请在此文档的“目录概览”小节补充说明，保持信息一致。
